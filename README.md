## 📄 Project Description

**Rootsy** is a thoughtfully designed website that embraces the essence of nature and eco-living. Perfect for eco-conscious brands, lifestyle blogs, or green businesses, this project emphasizes clean design and intuitive navigation. Built using HTML, CSS, and modern web standards, Rootsy delivers a seamless experience on both desktop and mobile. With fast loading speeds and natural-themed visuals, it invites users into a calming digital space that promotes harmony with the environment.

Whether you're building a green business, personal blog, or an awareness campaign — Rootsy offers a beautiful foundation.
